# French Recipes

## Classic French Cuisine

### 1. [Coq au Vin](coq-au-vin.md)
Chicken slowly braised in red wine with bacon, mushrooms, and pearl onions. A rustic Burgundy classic.

### 2. [Beef Bourguignon](beef-bourguignon.md)
Tender beef braised in red wine with vegetables and aromatics. An iconic Burgundy stew.

### 3. [Ratatouille](ratatouille.md)
Vibrant Provençal vegetable stew with eggplant, zucchini, peppers, and tomatoes.

### 4. [Quiche Lorraine](quiche-lorraine.md)
Classic French tart with buttery crust filled with rich custard, bacon, and cheese.

### 5. [Bouillabaisse](bouillabaisse.md)
Traditional Provençal fish stew from Marseille with saffron-infused broth and rouille.
