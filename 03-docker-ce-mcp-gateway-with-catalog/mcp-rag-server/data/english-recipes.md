# English Recipes

## Classic English Cuisine

### 1. [Fish and Chips](fish-and-chips.md)
Britain's most iconic dish with crispy beer-battered fish and thick-cut fried potatoes.

### 2. [Shepherd's Pie](shepherd-pie.md)
Hearty British classic with savory minced lamb in rich gravy topped with creamy mashed potatoes.

### 3. [Beef Wellington](beef-wellington.md)
Elegant showstopper with beef tenderloin, mushroom duxelles, and pâté wrapped in puff pastry.

### 4. [Bangers and Mash](bangers-and-mash.md)
British pub classic with savory sausages over creamy mashed potatoes with rich onion gravy.

### 5. [Sticky Toffee Pudding](sticky-toffee-pudding.md)
Decadent dessert with moist date sponge cake drenched in rich toffee sauce.
